## Description

Apply discounts based on specific conditions in WooCommerce.

## Features

- Apply discounts based on product categories
- Set discounts by user roles
- Create conditions for cart totals
- Support for percentage and fixed discounts
- Easy-to-use admin interface

## Installation

1. Download the plugin from your WordPress admin dashboard
2. Navigate to **Plugins > Add New**
3. Click **Upload Plugin** and select the `.zip` file
4. Click **Install Now** and then **Activate**

## Usage

1. Go to **WooCommerce > Conditional Discounts**
2. Click **Add New Discount**
3. Set your conditions (product category, user role, cart total, etc.)
4. Choose discount type (percentage or fixed amount)
5. Save and activate the discount

## Requirements

- WordPress 5.0 or higher
- WooCommerce 4.0 or higher
- PHP 7.4 or higher
