Apply conditional discounts to WooCommerce products based on custom rules.

== Description ==

Conditional Discount is a WooCommerce plugin that allows store owners to create and manage flexible discount rules. Set discounts based on product categories, quantities, user roles, and more.

== Features ==

* Create multiple discount rules
* Set discounts by product category
* Quantity-based discounts
* User role-based discounts
* Easy-to-use admin interface

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/` directory
2. Activate the plugin through the WordPress admin panel
3. Navigate to the plugin settings to create discount rules

== Frequently Asked Questions ==

= How do I create a discount rule? =

Go to the plugin settings in the WordPress admin and click "Add New Rule". Configure your conditions and discount amount.

= Can I apply multiple discounts? =

Yes, you can create multiple rules. The plugin will apply applicable discounts based on your configuration.
